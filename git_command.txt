git init - يستخدم لتهيئة مجلد كمستودع Git جديد

git status - يعرض حالة الملفات المعدلة و الجديدة في المستودع

git add filename - يضيف الملف المحدد إلى قائمة التغييرات الجاهزة للـ commit

git add . - يضيف جميع الملفات المعدلة والجديدة إلى قائمة الـ commit

git commit -m "رسالة" - يحفظ التغييرات مع كتابة رسالة توضح ما تم تعديله
